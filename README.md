# nextjs-boilerplate

This repository is for dogu's web-front nextjs boilerplate.

## How to use?

```sh
npx create-next-app --example [dogu-repo-url]
```

## What's in here?

1. Next.js
2. next-translate
3. styled-components
4. eslint, prettier
5. tsconfig
6. prerun script
7. default env(`DOGU_RUN_TYPE`, `NEXT_PUBLIC_ENV`)
8. axios
